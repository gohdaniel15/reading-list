# Setup

1. Clone the directory
```
git clone https://github.com/gohdaniel15/reading-list.git
```

2. Change directory
```
cd reading-list
```

3. Bundle
```
bundle install
```

4. Set up database
```
rake db:create db:migrate db:seed
```

# Run the site locally

1. Run the server
```
rails s
```

2. Navigate to localhost:3000
